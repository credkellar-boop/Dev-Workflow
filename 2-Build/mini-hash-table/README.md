# Custom Hash Table from Scratch

A bare-metal hash table data structure implementation built using standard Python to explore algorithmic memory layouts.

## 🛠️ Low-Level Architecture
* **Custom Polynomial Hashing:** Processes incoming keys through an ASCII-value multiplier loop to construct distributed index footprints.
* **Separate Chaining Collision Management:** Uses a nested dynamic array layout to maintain isolated collection paths when diverse data keys map to overlapping indices.
